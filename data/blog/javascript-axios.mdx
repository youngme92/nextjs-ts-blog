## 목적

Axios-mock-adapter는 단위 테스트 또는 로컬 개발 환경에서 Axios로 만든 HTTP 요청을 제어하기 위한 유틸리티입니다. 
요청을 가로채고 가짜 응답을 반환하므로 실제로 실제 HTTP 요청을 하지 않고도 애플리케이션을 테스트할 수 있습니다.

서버 API가 준비되지 않았을때 mock API 를 세팅하여 개발환경에서 사용할 수 있습니다.

## 과정

```shell
 axios-mock-adapter --save-dev
```

```js
import axios from 'axios';
import MockAdapter from 'axios-mock-adapter';

const mock = new MockAdapter(axios);
```

```js
mock.onGet('/users').reply(200, {
  users: [
    { id: 1, name: 'John' },
    { id: 2, name: 'Jane' },
  ],
});

```

```js
axios.get('/users').then((response) => {
  console.log(response.data);
});
```
