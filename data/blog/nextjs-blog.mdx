## 목적

개발일지 기록, 정보공유를 위해서 개발블로그를 만들기로 결정하였다.

그리고 개발블로그가 좋은 포트폴리오가 될수가 있기에 더더욱 만들어야 했다. 😇 

NextJs를 선택하게 된 이유는 요즘 핫한 기술이기도 하고, 정적페이지인 블로그 프로젝트에 적합할 것 같았다.

## 과정

<img width="1305" alt="스크린샷 2023-04-20 오전 11 09 30" src="https://user-images.githubusercontent.com/50010735/233239693-08aa36de-b5e0-4ddb-8c26-73b0b2a47545.png" />

vercel 에서는 nextJs를 활용한 여러 템플릿들을 지원한다. 그중  [Tailwind CSS Blog](https://vercel.com/templates/next.js/tailwind-css-starter-blog) 템플릿을 사용해보았다.



Blog 사이트는 생성하는 방법은 아래와 같다.

```bash
npm i -g @pliny/cli
```
```bash
pliny new --template=starter-blog my-blog
```
```bash
npx degit 'timlrx/tailwind-nextjs-starter-blog#contentlayer'
```

위 순서 대로 실행하면 블로그를 단번에 만들 수 있다.

### 이후에 추가로 개발 예정항목

댓글기능과, 뉴스레터 구독기능 등, 블로그에 필요한 요소를 하나씩 추가로 개발할 예정이다.
