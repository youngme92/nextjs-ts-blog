## 목적

개발일지 기록, 정보공유를 위해서 개발블로그를 만들기로 결정하였다.

그리고 개발블로그가 좋은 포트폴리오가 될수가 있기에 더더욱 만들어야 했다. 😇 

NextJs를 선택하게 된 이유는 요즘 핫한 기술이기도 하고, 정적페이지인 블로그 프로젝트에 적합할 것 같았다.

## 과정
Blog 사이트는 `*Pliny`를 사용하여 제작하였다.

```bash
npm i -g @pliny/cli
```
```bash
pliny new --template=starter-blog my-blog
```
```bash
npx degit 'timlrx/tailwind-nextjs-starter-blog#contentlayer'
```

위 순서 대로 실행하면 블로그를 단번에 만들 수 있다.

### 구현해보면서 알게 된 내용을 정리

Pilny 모듈은 `nextjs`과 `tailwind`기반으로 `markdown/mdx` 파일을 쉽게 관리 할 수 있었다. 

파일 마다 모듈을 사용한 부분이 많이 보였고 그 부분을 자세히 파악하진 못했지만 문서나 영상을 보니 블로그와 같은 개발에 유용해 보였다.
