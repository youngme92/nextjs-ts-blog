## 목적

지난 시간에 PlanetScale 세팅을 해보았다.

PlanetScale은 terminal 에서 git을 쓰듯이 디비에 접속하여 수정 및 생성 할 수 있고, DB툴에 연결해서 사용할 수도 있다.

바로 확인해 보자

## 과정

```bash
pscale auth login
```

우선 로그인을 진행한다.

위 명령어를 입력하면 계정을 확인하는 웹이 열리고 확인버튼을 누르게 되면 로그인이 완료되었다는 문구가 나타난다.

```bash
pscale shell <project-name> main
```

그리고 나서 프로젝트에 접속하면 DB 접속완료!

이제 명령어를 통해 SQL문을 사용할 수있다.

### DB 툴에 연결하는 방법

```bash
pscale connect <project-name> main --port 3309
```
port 는 자유롭게 입력해도 된다.

위 명령어를 입력하면
