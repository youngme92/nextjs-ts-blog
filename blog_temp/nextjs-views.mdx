## 목적

busuanzi 라는 오픈소스를 활용하여 개발 블로그에 사이트와 페이지 조회수를 표시하려고 한다.

## 과정

아래 코드를 body 혹은 head에 추가한다.

```js
<script async src="//busuanzi.ibruce.info/busuanzi/2.3/busuanzi.pure.mini.js"></script>
```

조회수를 보여줄 페이지에 아래 코드를 삽입한다.

### 사이트 조회수

```js
<span id='busuanzi_value_site_pv' ></span> 회
```

### 사이트 방문자
```js
<span id='busuanzi_value_site_uv' ></span> 명
```

### 페이지 조회수
```js
<span id='busuanzi_value_page_pv'></span> 회
```

## 결과

<div id="busuanzi_container_site_pv">이 사이트의 총 조회 횟수<span id="busuanzi_value_site_pv"></span>회</div>
<div id="busuanzi_container_site_uv">이 사이트의 총 방문 횟수<span id="busuanzi_value_site_uv"></span>회</div>
<div id="busuanzi_container_page_pv">이 페이지 조회 수<span id="busuanzi_value_page_pv"></span>회</div>
